# Use Case 1

## Research papers general reading


Given a path name, this path have many pdfs files downloaded from arXiv. and google scholar.


The user want to read all the papers in this path, and get a summary of each paper, group them into different categories based on their topics.


This use case will require DocMeld to:
- Extract text from PDFs (bronze layer)
- Summarize each paper (silver layer)
- Group papers into categories based on their topics (gold layer, extract topics by deepseek-chat model)
- Move the pdf papers and its bronze folder to a new folder based on the category
